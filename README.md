# 🤖 TutorBOT — AI-ассистент для репетиторов

Автоматическая генерация персональных отчётов для родителей учеников за 30 секунд.  
Снижает операционные расходы на 99% и освобождает ~10 часов работы репетитора в месяц.

## 🚀 Функции
- ✅ Генерация тёплых, профессиональных отчётов через Telegram
- 🧠 RAG-архитектура: поиск данных ученика в векторной БД
- 🛡️ Автоматическое смягчение резких формулировок (этика отчётов)
- 📊 Телеметрия и логирование всех сессий
- 🔒 Локальный запуск: нулевая стоимость токенов, полная приватность

- ## 🧠 Архитектура агента

1. Пользователь отправляет запрос в Telegram
2. `intent_detector.py` определяет тип задачи (отчёт / вопрос / планирование)
3. `rag_agent.py` извлекает контекст ученика из ChromaDB
4. LLM генерирует ответ с учётом этических правил (`guardrails.py`)
5. `sanitizer.py` проверяет вывод на утечки данных
6. Ответ отправляется пользователю, сессия логируется в `telemetry.py`

[Схема: можно добавить простую блок-схему в Mermaid или картинку]

## 🛠 Технологический стек
- **Язык:** Python 3.10+
- **Интерфейс:** Telegram Bot (aiogram 3.x)
- **LLM:** Qwen2.5:7b-instruct (локально через Ollama)
- **RAG/Память:** ChromaDB + all-MiniLM-L6-v2
- **Безопасность:** Input/Output Guardrails, санитайзер данных
- **Инфраструктура:** Локальный сервер / VPS

## ⚙️ Запуск проекта
1. Клонируйте репозиторий: `git clone https://github.com/ваш-ник/tutorbot-agent.git`
2. Создайте окружение: `python -m venv venv && source venv/bin/activate` (или `venv\Scripts\activate` на Windows)
3. Установите зависимости: `pip install -r requirements.txt`
4. Установите Ollama и модель: `ollama pull qwen2.5:7b-instruct`
5. Скопируйте `.env.example` в `.env` и добавьте токен Telegram-бота
6. Запустите: `python telegram_bot.py`
